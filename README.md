# GitHub User Finder

GitHub profile explorer app to search users and view their repositories with stats.

**Live Demo:** [https://github-user-finder-two-nu.vercel.app](https://github-user-finder-two-nu.vercel.app)

## Features
- Search any GitHub user by username
- View user profile info (followers, following, repos count)
- Display user profile photo
- Show recent repositories with details
- Display stars and forks count for each repo
- View repository descriptions
- Fully responsive design
- Built with React, Tailwind CSS, GitHub API

## Tech Stack
- React
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3
- GitHub API
